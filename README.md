## 

本着降低开发门槛的目的，我尝试了摒弃各种前端工具，甚至是HTML、CSS的编写

对项目的js文件进行模块化管理，特别是内置的layout布局/munu菜单/router路由等模块，可以快速生成基础SPA项目

可以说，她是为服务端开发人员量身定做，只需专注于JS的配置，从而达到由数据去驱动渲染，让你的页面更加信手捏来
